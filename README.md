# Ready Coast Prep V5 Fixed

This build starts from the last known working V3 application and applies the new branding without external logo dependencies.

## Upload correctly

Open this folder and upload the files inside it directly to the root of your GitHub repository. Do not upload the enclosing `ready_coast_v5_fixed` folder itself.

The repository root should visibly contain:

- index.html
- app.js
- styles.css
- sw.js
- manifest.webmanifest
- icon.svg
- icon-192.png
- icon-512.png

After GitHub deploys, open:

`https://readycoastprep.com/?v=5`

Then press Ctrl+Shift+R once. On a phone, remove the previously installed app and reinstall it after the website shows the new navy branding.
